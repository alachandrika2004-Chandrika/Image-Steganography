Image Steganography Using C
Project Overview

This project implements image steganography using C programming to hide and extract secret data inside image files. It demonstrates file handling, encoding/decoding logic, and basic data security concepts. The project is hosted on GitHub.

Features

Hide secret text/data inside an image file

Extract hidden data from the encoded image

Encoding and decoding functionality

Implementation using C file handling

Technologies Used

C Programming

File Handling Concepts

Basic Data Security Techniques

How to Run

Compile the program

gcc *.c


Run encoding

./a.out -e beautiful.bmp secret.txt output.bmp


Run decoding

./a.out -d stego.bmp extracted.txt

Learning Outcomes

Practical experience with C file operations

Understanding steganography basics

Improved logical thinking and debugging skills
